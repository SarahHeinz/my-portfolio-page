GitHub Repo Gallery

I used JavaScript and APIs (Application Programming Interface) to create a gallery of repositories. A repository is a place where you can store your code. You can store your files, and each file's revision history. All this information is stored in the GitHub cloud. You can click on a repository in the gallery. Then the data is pulled from the GitHub cloud because the API connects the two applications. Now you can see more information about the repository.

These are my skills: "JavaScript, APIs, Manipulate the DOM"

I use these tools: "Visual Studio Code, GitHub"

Guess the Word Game

This project is a game. You can guess one letter each round until you find out what the hidden word is. You have a maximum of 8 opportunities to find out the word. If you guess the word correctly you win the game. I used Javascript to write the game. You can play the game on my GitHub page.

These are my Skills: "Vanilla JavaScript, Manipulate the DOM"

I use these Tools: "Visual Studio Code, GitHub"

Unplugged-Website

I used CSS3 and HTML5 to code a website for a retreat. The website has 3 pages. First there is the main page. Second there is an informative page about the retreat. Third there is a page with frequently asked questions about the retreat. I used responsive web design. This means that the web page will look good on a mobile phone, a tablet and a laptop.

These are my skills: "CSS3, Flexbox, HTML5, Media Queries, Responsive Typography"
