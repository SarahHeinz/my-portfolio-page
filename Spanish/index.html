Generador de etiquetas

Con este Generador puedes preparar tu próxima reunión facilmente. Coja su lista de nombres e introdúzcalos en el campo de entrada. Ya tendrás las etiquetas con el nombre de todo el mundo en un segundo. He utilizado JSX, functional components y módulos de ES6 para crearlo.

Habilidades: "ES6, JSX, Managing data flow, React"

Galería de Repos de GitHub

He utilizado JavaScript y APIs (interfaz de programación de aplicaciones) para crear una galería de repositorios. En un repositorio puedes guardar tu código, tus carpetas y la historia de cambios de tu código. Toda esta información se guarda en la nube de GitHub. Cuando haces clic en cualquier repositorio en la galeria la API contecta las dos aplicaciones y extrae los datos. Así puedes ver más información sobre el repositorio.

Habilidades: "JavaScript, APIs, Manipulación del DOM"

Herramientas: "Visual Studio Code, GitHub"

"Adivina la Palabra"-Juego

En este proyecto he usado JavaScript para crear un juego. Puedes adivinar una letra cada turno hasta qué adivinas la palabra escondida. Tienes 8 intentos para adivinar la palabra correcta y ganar. Puedes jugarlo en mi página de GitHub. Disfruta!

Habilidades: "Vanilla Javascript, Manipulate the DOM"

Herramientas: "Visual Studio Code, GitHub"

Unplugged-Página web

Programé con CSS3 y HTML5 una página web para un retiro. La página contiene 3 páginas: primero la página principal, segundo una página con informaciones adicionales y tercero una con las preguntas frecuentes. La página es "responsive". Eso significa qué se ve igual de genial en el móvil, cómo en la tablet y el ordenador.

Habilidades: "CSS3, Flexbox, HTML5, Media Queries, Responsive Typography"

Herramientas: "Visual Studio Code, GitHub, Text Editor, Photoshop"

¡Hola! Soy Sarah Heinzkyll.

Me encanta crear sitios web y aplicaciones accesibles y fáciles de usar para los proyectos qué te apasionan y tus utopías sociales.

Cuando era una niña, me gustaba jugar con el ordenador. Me maravillaban todas sus posibilidades. Luego mi camino me llevó hacia otros intereses. Durante años trabajé en organizaciones que se movilizan contra la discriminación y por más justicia social. También quieren apoyar a los supervivientes de las violencias. Ese trabajo era con jóvenes y adultos en Berlín. En 2019 me mudé a un pequeño pueblo de España. Aquí he retomado mi interés por las tecnologías. ¿Tienes grandes ideas? ¿Necesitas un sitio web o una aplicación para hacerlas realidad? Estoy encantada de ayudar a tu empresa, organisación o movimiento de base a difundir su mensaje en el mundo virtual. También me encanta trabajar con artesanes creatives ayudándoles a visibilizar sus artes en Internet.

Cuéntame tus ideas y sueños. ¡Hagámoslas realidad!

Me alegra saber de ti!
