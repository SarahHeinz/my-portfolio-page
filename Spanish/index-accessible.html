Generador de etiquetas de nombres

Con este Generador puedes preparar tu próxima reunión facilmente. Coge tu lista de nombres. Ahora introdúzcalos en el campo de entrada. Ya tendrás las etiquetas con el nombre para todes en un segundo. He utilizado JSX, "functional components" y módulos de ES6 para crearlo.

Esas habilidades necesité: "ES6, JSX, Gestionar el flujo de datos, React"

Esas herramientas utilicé: "CodeSandbox"

Galería de Repos de GitHub

He utilizado JavaScript y APIs (interfaz de programación de aplicaciones) para crear una galería de repositorios. En un repositorio puedes guardar tu código. Puedes guardar tus carpetas y la historia de cambios de tu código. Toda está informacion se guarda en la nube de GitHub. Puedes hacer clic en cualquier repositorio en la galeria. La API permite contectar las dos aplicaciones y extraer los datos. Ahora puedes ver más información sobre el repositorio.

Esas habilidades necesité: "JavaScript, APIs, Manipulación del DOM"

Esas herramientas utilicé: "Visual Studio Code, GitHub"

Unplugged-Página Web

Programé con CSS3 y HTML5 una página web para un retiro. La página contiene 3 páginas: primero la página principal. Segundo hay una página con informaciones adicionales sobre el retiro. Y hay un parte de preguntas frecuentes. La página es "responsive". Eso significa qué se ve igual de bueno en el móvil, cómo en la tablet y el ordenador.

Esas habilidades necesité: "CSS3, Flexbox, HTML5, Media Queries, Responsive Typography"

Esas herramientas utilicé: "Visual Studio Code, GitHub, Text Editor, Photoshop"

Hola, soy Sarah Heinzkyll.

Me encanta crear páginas web y aplicaciones accesibles y fáciles de usar para los proyectos qué te apasionan. Especialmente si buscan crear justicia social.

Cuando era una niña, me gustaba jugar con el ordenador. Me maravillaban todas sus posibilidades. Luego mi camino me llevó hacia otros intereses. Durante años trabajé en organizaciones que se movilizan contra la discriminación y por más justicia social. También quieren apoyar a los supervivientes de las violencias. Mi trabajo era con jóvenes y adultos en Berlín. En 2019 me mudé a un pequeño pueblo de España. Aquí he retomado mi interés por las tecnologías. ¿Tiene grandes ideas? ¿Necesita un sitio web o una aplicación para hacerlas realidad?
