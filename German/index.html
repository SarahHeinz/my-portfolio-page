Notizzettel-App

Mit dieser App können Sie virtuelle Notizzettel schreiben. Haben Sie einen Fehler gemacht? Kein Problem. Die Aufgabe wurde erledigt? Super! Die Notizen lassen sich jederzeit editieren, löschen und Sie können sie auch mit Hilfe von Stichwörtern durchsuchen. Diese App wurde mit React.JSX geschrieben.

Skills: "ES6, JSX, React, DOM Manipulation"

"GitHub Repo"-Gallerie

Ich habe JavaScript und APIs (Application Programming Interface) verwendet, um eine "Repository"-Galerie zu kreieren. Ein Repository ermöglicht es Ihnen, Ihren Code, Ihre Dateien und den Revisionsverlauf jeder Datei in der GitHub-Wolke zu speichern. Wenn Sie auf ein Repository in der Galerie klicken, verbindet die API die beiden Anwendungen und die Daten werden abgerufen, sodass Sie weitere Informationen über das Repository sehen können.

Skills: "JavaScript, APIs, DOM Manipulation"

Tools: "Visual Studio Code, GitHub"

Unplugged-Webseite

Ich habe mit Hilfe von CSS3 und HTML5 eine Webseite für einen Retreat programmiert. Diese Webseite besteht aus drei Seiten: der Hauptseite, einer Seite mit weiterführender Information zum Retreat und einer Seite mit häufigen Fragen. Die Webseite ist "responsive", was bedeutet, dass sie auf Laptop, Handy und Tablet perfekt aussieht.

Skills: "CSS3, Flexbox, HTML5, Media Queries, Responsive Typography"

Tools: "Visual Studio Code, GitHub, Text Editor, Photoshop"

Dann veränderte sich mein Fokus und ich arbeitete mehrere Jahre in Berlin in verschiedenen Anti-Gewalt-Organisationen, in denen wir für soziale Gerechtigkeit mobilisierten und Überlebende von Gewalt unterstützten. 2019 bin ich in ein Dorf in Spanien gezogen, und dort auch wieder zu meiner Tech-Leidenschaft zurückgekehrt.
