Notizzettel-App

Mit dieser App können Sie virtuelle Notizzettel schreiben. Haben Sie einen Fehler gemacht? Das ist kein Problem. Die Aufgabe wurde erledigt? Das ist super! Sie können die Notizen jederzeit verändern oder löschen. Sie können auch Stichwörter in den Notizen suchen. Diese App wurde mit React.JSX geschrieben.

Skills: "ES6, JSX, React, DOM Manipulation"

"GitHub Repo"-Galerie

Mit Hilfe von JavaScript und APIs (Application Programming Interface) habe ich eine "Repository"-Galerie gemacht. Mit einem Repository können Sie Ihren Code in der GitHub Cloud speichern. Klicken Sie auf ein Repository in der Galerie. Nun verbindet die API die beiden Anwendungen. Die Daten werden abgerufen und Sie sehen weitere Informationen über das Repository.

Meine Fähigkeiten: "JavaScript, APIs, Dom Manipulation"

Technische Werkzeuge: "Visual Studio Code, GitHub"

Unplugged-Webseite

Ich habe mit Hilfe von CSS3 und HTML5 eine Webseite für einen Retreat programmiert. Diese Webseite besteht aus drei Seiten. Erstens gibt es die Hauptseite. Zweitens gibt es eine Seite mit weiterführender Information zum Retreat. Drittens gibt es eine Seite mit häufigen Fragen über den Retreat. Die Webseite ist "responsive". Das bedeutet, dass sie auf dem Laptop, Handy und dem Tablet perfekt aussieht.

Meine Fähigkeiten: "CSS3, Flexbox, HTML5, Media Queries, Responsive Typography"

Technische Werkzeuge: "Visual Studio Code, GitHub, Text Editor, Photoshop"
